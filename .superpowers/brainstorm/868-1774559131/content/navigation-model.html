Navigation Model

How should the user move through the memory loop dashboard?

A

Tab-Based Panels

Top-level tabs switch between views: Memories (filterable table), Injection Preview (test prompts), History (refresh & injection logs), Settings (mode/policy). Arrow keys or number keys to switch tabs.

B

Master-Detail Split

Left panel shows the memory list (always visible). Right panel shows detail of the selected memory — body, evidence, history, injection stats. Actions apply to the focused memory. Filter/status tabs sit above the list.
