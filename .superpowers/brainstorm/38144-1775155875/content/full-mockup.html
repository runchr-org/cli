Key Behaviors

  • No Enter required — detail pane updates instantly as you move through the list with j/k
  • Filters: 1 cycles TIME (24h → 7d → 30d → all), 2 cycles BRANCH (repo → current → all)
  • Selected row highlighted with amber background + left accent border
  • Detail sections are bordered boxes: SESSION metadata, SUMMARY, FRICTION, LEARNINGS, INSIGHTS
  • Detail scrolls with j/k when list isn't focused (or always scrolls if content overflows)
  • g key generates summary/facets for sessions that don't have them yet
  • Pagination still available with ←→ for the list
