Filter Bar Design

How should multi-dimension filters be presented? Press 1/2/3 to switch dimensions, f to cycle values.

A

Inline Tab Row

All three filters visible at once in a single row. Active values highlighted. Press number key to focus a filter, then f to cycle its values.

TIME: today │ week │ month │ all BRANCH: repo │ current │ all CHECKPOINT: with │ without │ all
1/2/3 select filter · f cycle value · 5 sessions
B

Compact Chip Row

Filters shown as compact chips/pills. Each chip shows the active value. Press number to cycle that chip.

1 TIME: today 2 BRANCH: all 3 CKPT: all 5 sessions
1/2/3 cycle filter · 5 sessions shown
C

Stacked: Active Filter Bar + Context Line

Top line shows which filter dimension is selected (cycle with Tab). Second line shows current state of all filters as a summary.

▸ TIME │ BRANCH │ CHECKPOINT
today · week · month · all showing: today │ all branches │ all checkpoints │ 5 sessions
Tab switch filter · f cycle value
